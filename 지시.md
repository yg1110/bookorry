# 부코리 (bookorry) - Next.js 북클럽 독후감 웹앱

## 프로젝트 스펙

- Next.js 14 (App Router)
- Tailwind CSS
- Supabase (PostgreSQL)
- 모바일 웹 위주
- 로그인 없음 — 초대코드 + 닉네임만으로 참여

## DB 구조

groups id / name / invite_code / created_at
members id / group_id / nickname / session_token / created_at
books id / group_id / title / author / thumbnail / registered_at
reviews id / book_id / member_id / content / created_at

## 지금까지 완료

- Next.js + Supabase + Tailwind 셋업 완료
- .env에 Supabase URL, ANON_KEY 설정 완료

## 지금 만들 것

그룹 생성 + 초대코드 참여 기능

### 필요한 작업

1. Supabase에 groups, members 테이블 생성 SQL
2. Supabase 클라이언트 설정 (lib/supabase.ts)
3. 홈 페이지 (app/page.tsx)
   - 그룹 만들기 버튼
   - 초대코드 입력 + 참여하기 버튼
4. 그룹 생성 페이지 (app/create/page.tsx)
   - 그룹명 입력
   - 닉네임 입력
   - 생성 시 랜덤 초대코드 발급 (예: BOOK-A1B2)
   - 생성 후 초대코드 표시 + 복사 버튼
5. 초대코드로 참여 페이지 (app/join/page.tsx)
   - 코드 확인 후 닉네임 입력
   - session_token을 localStorage에 저장해서 멤버 식별

모바일 퍼스트 디자인으로 만들어줘.
